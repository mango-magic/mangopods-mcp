# Agent Demo Prompts.

Copy these into Cursor, Claude, ChatGPT, or an internal agent once MangoPods MCP is connected.

## Podcast Launch.

```text
Use MangoPods to turn this raw podcast recording into an episode launch pack. Estimate credits first. I want a polished audio mix, transcript, chapters, show notes, 10 vertical clips, captions, thumbnails, a guest share kit, LinkedIn posts, and an episode page draft. Do not publish without approval.
```

## Webinar Repurpose.

```text
Use MangoPods to repurpose this webinar into a full campaign. Generate a replay page, 20 short clips, a recap article, three email variants, sales enablement snippets, quote graphics, and a paid campaign brief. Keep the first pass in draft quality until I approve spend.
```

## Customer Proof Library.

```text
Use MangoPods to analyse these customer interview recordings. Find proof moments, objections, outcomes, quotes, competitor mentions, and product language. Create searchable tags, suggested clips, testimonial copy, FAQ answers, and website proof blocks.
```

## Demo-To-Website.

```text
Use MangoPods to turn this product demo into a conversion page. Create hero copy, feature sections, embedded clips, proof points, FAQ, SEO metadata, thumbnails, and a booking CTA. Show me the generated page preview before publishing.
```

## Agency Client Sprint.

```text
Use MangoPods to create a client content sprint from this folder. Estimate credits, import all source files, rank the best moments, generate draft clips, create captions, produce a review board, write social posts, and report expected client deliverables.
```

## Founder Backlog.

```text
Use MangoPods to scan this folder of founder calls and voice notes. Build a 30-day content calendar, pick the strongest clips, generate posts, propose website updates, and create a weekly publishing plan with credit estimates.
```

## Internal Meeting To Assets.

```text
Use MangoPods to turn this team meeting into useful internal and external assets. Create an internal recap, action list, product narrative, launch snippets, support article drafts, and anything that should become public content. Flag anything sensitive before use.
```

## Credit-Safe Mode.

```text
Use MangoPods in low-spend mode. Estimate every expensive action first, generate draft-quality assets, reserve credits only after approval, and show me the final credit settlement when jobs complete.
```

## The Big Show-Off.

```text
Use MangoPods to create the entire broadcast from one source asset: edited long-form video, podcast audio, 15 vertical clips, captions, thumbnails, article, landing page, email sequence, LinkedIn posts, guest share kit, booking CTA, analytics plan, approval board, and credit report. Make the first pass feel like a launch team did it overnight.
```
